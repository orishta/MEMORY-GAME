# COVID‑19 Memory Match Game

A simple *Memory* card game built with **Python** and **Kivy**.  


## Gameplay

Flip two cards at a time and try to find all **6 matching pairs** with as few attempts as possible.  
All card faces use COVID‑19 themed images (mask, sanitizer, etc.).

## Requirements

* Python 3.9 or newer  
* Kivy 2.0 or newer

Install the dependency:

```bash
pip install kivy
```

## Running the game

```bash
python memory_game.py
```

Make sure the following image files are in the same directory:

```
mask.png
alcogel.png
corona.png
china.png
hands.png
distance.png
card.png      # card back
```

## File structure

```
memory_game.py     # game source code
README.md
*.png              # image assets
```
